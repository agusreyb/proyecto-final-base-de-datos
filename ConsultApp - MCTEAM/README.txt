======================================
MCTEAM - 5 - TP FINAL - ConsultApp
======================================

Programa GUI para facilitar al usuario para generar consultas sin necesidad de usar un sistema de gestor de Bases de datos.

El programa utiliza Python y las siguientes librerías:
psycopg2
pandas 
customtkinter 
tkinter (pip install tk)
pandasgui 
configparser

Puede instalarlos con "pip install [nombre modulo]" en el cli

ES NECESARIO QUE ESTE CON EL ARCHIVO "config.ini" EN EL MISMO DIRECTORIO QUE EL ConsultApp.py. Ademas, se requiere configurar ese mismo archivos con los parámetros de configuración de acceso a su base de datos.

Disfruten!
